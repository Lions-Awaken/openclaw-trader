What is Parallax?

▼

Parallax is an autonomous AI swing trading system running 24/7 on ridley — a NVIDIA Jetson Orin Nano 8GB. It runs a multi-profile architecture: the active profile is CONGRESS_MIRROR, which dynamically builds its watchlist from recent high-signal congressional trade disclosures (last 21 days). When no qualifying congressional filings exist, it falls back to a technical watchlist of liquid universe stocks.

Every candidate is evaluated through a 5-tumbler Lock & Tumbler inference engine: a staged reasoning chain that begins with free local computation (T1–T2), advances through a local GPU LLM call (T3 via Ollama qwen2.5:3b), and only reaches the paid Anthropic API for pattern matching (T4 Claude Haiku) and counterfactual synthesis (T5 Claude Sonnet) when confidence warrants it. Each tumbler has a confidence floor; falling below it stops the chain early.

The system is built around a 6-shadow adversarial ensemble: one live profile makes trading decisions, while six independent shadow agents (SKEPTIC, CONTRARIAN, REGIME_WATCHER, OPTIONS_FLOW, FORM4_INSIDER, KRONOS_TECHNICALS) silently evaluate every candidate and record their disagreements. Those disagreements become training data for a weekly self-calibrating feedback loop that learns which agents are consistently right and adjusts their DWM weights accordingly.

The system is conservative by design. When in doubt, it does nothing. Cash is a position.

The Scanner Pipeline — T1 through T5 Tumbler Chain

▼
T1 Technical Foundation FREE

Watchlist tickers scored against 6 binary signals: Trend, Momentum, Volume, Catalyst, Sentiment, Flow. RAG retrieves similar past signal evaluations via pgvector.

  • Score 0–6. Only tickers scoring ≥ 3 advance to the tumbler chain.
  • Pure local computation — no LLM inference involved.
  • Confidence floor: 0.25 — chain stops if not reached.
T2 Fundamental + Sentiment (RAG) FREE
  • Retrieves similar past inference chains and trade learnings via pgvector similarity search
  • Congressional trade disclosures (CONGRESS_MIRROR) get a confidence boost if filed within 40 days
  • Hard VETO if sentiment score < −0.5 — chain terminates immediately
  • Confidence floor: 0.40
T3 Flow & Cross-Asset (Ollama on ridley GPU) FREE (local)
  • First LLM call — runs locally on ridley's Jetson GPU via Ollama (qwen2.5:3b)
  • Analyzes how this setup compares to past chains and outcomes. Shadow context injected for shadow profiles.
  • Confidence adjustment: ±0.10
  • Confidence floor: 0.55 — STOP if delta < 0.03 (forced connection rule)
T4 Pattern Template Matching (Claude Haiku — Anthropic API) ~$0.001/call
  • Matches against 24 seeded pattern templates with documented outcomes via pgvector RAG
  • Evaluates quality and coherence of the full trade thesis
  • Confidence adjustment: ±0.10
  • Confidence floor: 0.65
T5 Counterfactual Synthesis (Claude Sonnet — Anthropic API) ~$0.005/call
  • Devil's advocate — constructs the strongest argument AGAINST the trade using trade_learnings loss RAG
  • Asymmetric adjustment: can drop confidence by 0.15 but raise it only 0.05
  • Calibration factor applied to final confidence
  • Has not yet fired in production (budget gating keeps it rare)
GATE Execution Gate & Decision Thresholds

Trade only executes if ALL THREE conditions are met:

  • Decision = enter or strong_enter
  • Final confidence ≥ 0.60
  • At least 3 tumblers completed

Decision thresholds: strong_enter ≥0.75   enter ≥0.60   watch ≥0.45   skip ≥0.20   veto <0.20

Stopping rules: veto_signal   confidence_floor   forced_connection   conflicting_signals   insufficient_data   resource_limit   time_limit (30s)

SKEPTIC LLM (Claude) Require overwhelming evidence. Confidence floors HIGH. Heavily penalizes momentum-chasing. Graded: Conditional Brier Score.
CONTRARIAN LLM (Claude) Assume the trade is wrong. Overweights sector rotation, institutional distribution, price-fundamental divergence. Graded: Regime-Conditional IC.
REGIME_WATCHER LLM (Ollama only, stops at T3) Ignore the ticker entirely. Only question: is this a good time to enter ANY long position? Evaluates SPY trend, VIX, yield curve, credit spreads. Zero API cost. Graded: signal latency.
OPTIONS_FLOW LLM (Claude) Primary signal: unusual options activity — sweeps, blocks, dark pool prints. Alpha decay 1–5 days. Graded: 5-day forward return.
FORM4_INSIDER LLM (Claude) Primary signal: Form 4 purchase filings by CEOs/CFOs/board members. Cluster buys weighted heavily. CFOs = strongest signal. Graded: 15-day forward return.
KRONOS_TECHNICALS GPU (Jetson, no LLM) Pure price pattern. 252 daily OHLCV candles → Kronos foundation model → 50 Monte Carlo paths → bullish probability at 10-day horizon. No news, no fundamentals. Local GPU only. Graded: 10-day directional accuracy.

The Calibration Loop — DWM

▼

Every Sunday, the calibrator grades all ungraded divergences from the past 30 days:

  1. Fetches ungraded shadow_divergences where shadow_was_right is null
  2. Grades each: was the shadow agent right to dissent?
    • Kronos: checks actual close price at 10 trading days after signal
    • LLM agents: checks inference_chains.actual_outcome
  3. Fitness score: fitness = times_correct / times_dissented
  4. DWM weight: new_weight = 1.0 × (1 + 0.5 × (fitness − median_fitness))  clamped [0.05, 3.0]

Meta Daily Reflection — 1:30 PM

▼

Every afternoon, Claude reviews the day's shadow divergences looking for unanimous dissent — all 6 shadows simultaneously disagreed with the live profile. If found:

1 Full divergence data injected into Claude Sonnet prompt as HIGH PRIORITY
2 Instruction: "Analyze what the shadows saw that the live profile missed"
3 Output written to meta_reflections table
4 The reflection becomes RAG context for future T2 tumbler calls

Circuit Breakers and Risk Limits

▼
  • 3 consecutive losses → circuit breaker trips, scanner stands down
  • Daily drawdown > 2% → circuit breaker trips (compares equity vs last_equity from Alpaca)
  • Max concurrent positions → configurable per strategy profile (default: 5); scanner scans but does not execute when slot limit reached
  • 5% max risk per trade (ATR-based position sizing)
  • Max hold period → configurable per strategy profile (default: 3 days); positions closed via time stop when exceeded
  • Confidence floor: 0.60 → no trade executes below this threshold regardless of decision label
